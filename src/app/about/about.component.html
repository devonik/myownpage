Über mich

Mich interessieren so ziemlich alle aktuellen Technologien, die im Web vertreten sind. Sei es ASP.Net MVC, .Net Core im Backend Bereich, Angular und Typescript im Frontend Bereich, oder etwas wie z.B. die stetig wachsenden VR Technologien.

Als Web Entwickler kann ich sehr gut sowohl im Team als auch selbstständig arbeiten. Ich entwickle meist Full-Stack, das heißt, ich entwickle sowohl das Frontend, das Backend als auch die Datenbank Funktionen. Meine Präferenzen liegen allerdings in der Frond- und/oder Backend Entwicklung. Zudem bin ich ein flexibler, motivierter und aufgeschlossener Kollege :)
